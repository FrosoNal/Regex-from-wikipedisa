# Wikipedia Data Extractor (NLP Project)

## Περιγραφή
Αυτό το πρόγραμμα αναπτύχθηκε για το μάθημα "Επεξεργασία Φυσικής Γλώσσας". Σκοπός του είναι η αυτόματη εξαγωγή πληροφοριών από σελίδες πόλεων της Wikipedia χρησιμοποιώντας αποκλειστικά κανονικές εκφράσεις (Regular Expressions).

## Τεχνολογίες
- Python (Standard Library: `re`, `urllib.request`)

## Δομή Project
- `src/`: Ο κύριος κώδικας του προγράμματος.
- `docs/`: Η αναλυτική αναφορά της άσκησης σε PDF.

## Αποτελέσματα
Το πρόγραμμα εξάγει επιτυχώς:
- Όνομα πόλης, Πληθυσμό, Έκταση, Συντεταγμένες, Υψόμετρο, Ζώνη ώρας και Θερμοκρασία.
