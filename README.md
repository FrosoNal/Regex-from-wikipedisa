# Wikipedia Data Extractor

## Description
This project focuses on the automatic extraction of information from Wikipedia city pages using only regular expressions (RegEx).

## Technologies
- Python (Standard Library: `re`, `urllib.request`)

## Project Structure
- `src/`: The main source code.
- `docs/`: The detailed report in PDF format.

## Results
The program successfully extracts:
- City name, Population, Area, Coordinates, Country, Altitude, Timezone, and Temperature(Ligh, Mean, Low).

## How to run
1. Clone the repository.
2. Navigate to the `src/` folder.
3. Run: `python main.py`
